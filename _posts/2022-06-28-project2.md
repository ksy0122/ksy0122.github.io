## 계산기 만들기 - 1
첫 프로젝트로 계산기를 만들어보았다. 처음에는 간단하게 두 수를 입력받아 사칙연산 계산을 할 수 있는 프로그램을 짰다. switch ~ case문과 if문을 사용해 두 가지 방식의 코드를 작성해 보았다.
<br>

**switch ~ case문**
<br>

```
#include <stdio.h>

void main()
{
   float a, b;
   char on, z;

   while (1)
   {
      printf("프로그램을 종료하시겠습니까? (Y/N) : ");
      scanf_s("%c", &on);
      getchar();

      if (on == 'y' || on == 'Y')
      {
         printf("프로그램을 종료합니다.\n");
         break;
      }
      else if (on == 'n' || on == 'N')
      {
         printf("연산자를 입력하세요. : ");
         scanf_s("%c", &z);
         getchar();

         printf("숫자 두 개를 입력하세요. : ");
         scanf_s("%f", &a);
         scanf_s("%f", &b);
         getchar();

         switch (z)
         {
         case '+':
            printf("%f + %f = %f\n", a, b, a + b);
            break;

         case '-':
            printf("%f - %f = %f\n", a, b, a - b);
            break;

         case '*':
            printf("%f * %f = %f\n", a, b, a * b);
            break;

         case '/':
            printf("%f / %f = %f\n", a, b, a / b);
            break;

         default:
            printf("0000.\n");
            break;
         }
      }
   }
}
  ```
